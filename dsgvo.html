ROS CLOUD DSGVO

Die Datenschutzerklärung von ROS (Fassung 21.01.2020) nach Vorgaben der Datenschutzgrundverordnung (EU) 2016/679 erklärt, welche Informationen gesammelt werden und wie diese verwendet werden.

Für zusätzliche Informationen bezüglich Datenschutz, kontaktieren Sie uns unter
support@ros-cloud.at .

Automatische Datenspeicherung

Bei einem Besuch der Webseite werden automatisch gewisse Informationen gespeichert. Darunter fallen

  • ● die URL
  • ● Browser und Browserversion
  • ● das Betriebssystem
  • ● IP-Adresse
  • ● Datum und Uhrzeit


Diese Daten werden als Logs gespeichert und können beim Vorliegen von rechtswidrigem Verhalten behördliche eingesehen werden.

Speicherung persönlicher Daten

Persönliche Daten, wie zum Beispiel Name, E-Mail-Adresse oder andere Angaben werden im Rahmen einer Registrierung über ein Formular an den Drittanbieter Google Cloud Services (Firebase) übergeben.

Weitere persönliche Daten, wie diverse Dokumente und Dateien, die im Rahmen der Nutzung an ROS-Cloud, durch einen dafür vorgesehenen Upload übergeben werden, sind ausschließlich durch den ursprünglichen Nutzer ersichtlich. Diese können jedoch bei einem Vorliegen von rechtswidrigem Verhalten behördlich eingesehen werden.

TLS-Verschlüsselung mittels HTTPS

Um Daten abhörsicher zu übertragen kommt eine TLS-Verschlüsselung zum Einsatz(Datenschutz durch Technikgestaltung Artikel 25 Absatz 1 DSGVO ). Dadurch kann der Schutz vertraulicher Daten sichergestellt werden. Die Nutzung von TLS kann an dem Schlosssymbol neben der URL, sowie an der Verwendung des “https” Schemas der URL erkannt werden.

Verwendung von Cookies

Cookies werden von ROS-Cloud ausschließlich zur Benutzererkennung des Speichersystems verwendet und dienen nicht zu Zwecken der Analyse oder sonstigem.

Firebase Datenschutzerklärung

Firebase wird genutzt um Accountdaten (E-Mail Adresse, Benutzername, Passwort, Name) zu speichern und zu verwalten. Die Datenschutzbestimmungen von Firebase können unter folgender URL eingesehen werden:
https://firebase.google.com/support/privacy/#data_protection

Benutzerdaten von Firebase können über das Verwenden des Account-Löschen-Buttons im Dashboard der Webseite gelöscht werden. Dabei werden aber auch alle Daten innerhalb von 30 Tagen von den ROS-Cloud-Servern gelöscht und können nicht mehr eingesehen werden.

Google Fonts Datenschutzerklärung

ROS-Cloud verwendet Google Fonts, ein Dienst für die Bereitstellung von Schriften. Für den europäischen Raum ist das Unternehmen Google Ireland Limited (Gordon House, Barrow Street Dublin 4, Irland) verantwortlich.
Es werden keine Cookies bezüglich dieses Dienstes in Ihrem Browser gespeichert. Die Dateien (CSS, Schriftarten/Fonts) werden über die Google-Domains fonts.googleapis.com und fonts.gstatic.com angefordert. Laut Google sind die Anfragen nach CSS und Schriften vollkommen getrennt von allen anderen Google-Diensten.

Die Datenschutzerklärung von Google kann unter folgender URL eingesehen werden:

https://policies.google.com/privacy?hl=de

Welche Daten werden von Google gespeichert?

Wenn Sie die Website von ROS-Cloud besuchen, werden Schriften über einen Google-Server nachgeladen. Durch diesen externen Aufruf werden Daten an Google-Server übermittelt. So erkennt Google auch, dass Sie bzw. Ihre IP-Adresse ROS-Cloud besucht.

Wie kann ich meine Daten löschen bzw. die Datenspeicherung verhindern?

Um diese Daten vorzeitig löschen zu können, müssen Sie den Google-Support auf https://support.google.com/?hl=de&tid=111619908 kontaktieren. Datenspeicherung verhindern Sie in diesem Fall nur, wenn Sie ROS-Cloud nicht besuchen.

Welche Daten grundsätzlich von Google erfasst werden und wofür diese Daten verwendet werden, können Sie auch auf https://www.google.com/intl/de/policies/privacy/ nachlesen.

JS-Delivr Datenschutzerklärung

JS-Delivr wird von ROS-Cloud genutzt, um diverse Stylesheets und Skripte aus einem CDN zu beziehen. Um den Dienst Bereitzustellen, übermittelt Ihr Browser eine HTTP-Anfrage, mit einigen Benutzerdaten, wie IP-Adresse, Browser, Betriebssystem und der geladenen Webseite. In der Datenschutzerklärung von JS-Delivr wird ausdrücklich darauf hingewiesen, dass das Unternehmen keine Cookies oder andere Tracking-Services verwendet.

Die Datenschutzerklärung von JS-Delivr kann unter folgender URL eingesehen werden: https://www.jsdelivr.com/terms/privacy-policy-jsdelivr-net

Fontawesome Datenschutzerklärung

Fontawesome wird von ROS-Cloud genutzt, um diverse SVG-Icons aus einem CDN zu beziehen. Um den Dienst Bereitzustellen, übermittelt Ihr Browser eine HTTP-Anfrage, mit einigen Benutzerdaten, wie IP-Adresse, Browser, Betriebssystem und der geladenen Webseite. Zusätzlich speichert Fontawesome um welches Icon es sich bei der Anfrage gehandelt hat.

Zu Analysezwecken können Log-Daten länger als die übliche Zeit von ein paar wenigen Wochen gespeichert werden.

Die Datenschutzerklärung von Fontawesome kann unter folgender URL eingesehen werden: https://fontawesome.com/privacy

Einsicht in persönlicher Daten

Alle persönlichen Daten (abgesehen von Logs) können über das Dashboard der Webseite eingesehen und gelöscht werden.Durch eine E-Mail an die Adresse support@ros-cloud.at ist es auch möglich diese Daten in Form eines Dokumentes übermittelt zu bekommen.
